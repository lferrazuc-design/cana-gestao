# 🌿 Gestão Muda de Cana

Sistema web de gestão de compra e venda de muda de cana-de-açúcar.

## Funcionalidades

- Cadastro e gestão de pedidos
- Status de cada pedido (negociando, OS aberta, contrato, etc.)
- Geração e edição de contratos
- Envio de contratos por e-mail
- Aba de Validação de contratos
- Dashboard com KPIs e gráficos
- Relatórios por vendedor e comprador
- Exportação CSV
- Login com usuário e senha

## Login padrão

- **Usuário:** `admin`
- **Senha:** `cana2026`

> Para alterar, edite o array `USERS` no início do arquivo `src/App.jsx`.

## Rodando localmente

```bash
npm install
npm run dev
```

## Build para produção

```bash
npm run build
```
